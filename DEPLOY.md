# 免费上线步骤

推荐先用 Netlify，最省事，不需要 Mac。

如果你想用 GitHub Pages，请看 `GITHUB_PAGES.md`。

## Netlify 免费发布

1. 打开 https://app.netlify.com/drop
2. 登录或注册免费账号。
3. 把 `rhode-catch-pwa.zip` 拖到页面里。
4. 等它生成一个 `https://...netlify.app` 网址。
5. 用 iPhone Safari 打开这个网址。
6. 点分享按钮，选择“添加到主屏幕”。

## 更新 App

以后我帮你改完代码后，再重新上传新的 `rhode-catch-pwa.zip`，Netlify 会更新网站。

## 费用

这个阶段可以免费：

- Netlify 免费套餐足够测试。
- Open-Meteo 天气 API 免费。
- NOAA 潮汐 API 免费。
- 不买域名就不用花钱。

以后如果要上 App Store，才需要 Apple Developer Program，每年 99 美元。
